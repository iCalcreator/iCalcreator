Testing iCalcreator get/edit/delete/select functions

All test scripts, run in a web server, uses a test file created in createComponent.php, saved in a file named "test.ics" (folder must be writeable for web server user) and tests parsing, reading, insert, edit and delete of components in the test file.
Click on the links below to execute tests. Use the browsers 'back'- button to return to this page. Open the linked files in a text editor to examine the php code.
